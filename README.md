# LLM_MCP_Agent
A professional-grade AI Agent implementation using Model Context Protocol (MCP) for dynamic tool discovery and execution. Built with Python and GapGPT (GPT-gpt-4o).
## 🚀 Overview

This server provides a single example tool:

**`sum_numbers(a, b)`** → returns the sum of two integers.

It serves as a simple starter template for building your own MCP tool servers.

---

## 📦 Features
- Lightweight MCP server
- JSON-formatted responses
- Simple function-to-tool decorator
- Easily extendable with new tools
